# Haveit
### 프로젝트 주제

- 만들고 싶은 습관을 생성하여 여러 사람과 같이 습관을 실천합니다.
- 같은 습관에 등록된 사람은 같은 타임라인을 공유하며 매일 습관을 실천 한 것에대해 인증합니다.
- Springboot와 백엔드 시스템을 공부하는것에 초점을 맞추기 위해 view 는 최소화하고 서버 부분에 집중합니다.

### 개발 환경

- Springboot 3.1.0
- JAVA 17
- MySQL
- H2 Database
- Spring Data JPA
- Spring Data REST
- Spring Security
- Lombok
- Intellij
- Github
